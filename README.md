## 🚀 How It Works

1. Enter a number.
2. Click the **Check Data** button.
3. JavaScript checks the number using the modulus (`%`) operator.
4. The result is displayed instantly.

---

## 💻 JavaScript Logic

```javascript
function checkData() {
    let number = document.getElementById("number").value;

    if (number % 2 === 0) {
        document.getElementById("outputData").innerHTML = "Even";
    } else {
        document.getElementById("outputData").innerHTML = "Odd";
    }
}
```
